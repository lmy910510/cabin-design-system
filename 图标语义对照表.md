# 座舱图标语义对照表

> **用途**：业务场景 → 推荐图标的映射表。AI 调用图标时**先查此表**，找不到再去 Figma 库搜索/穷举。
>
> **数据源**：图标本体在 Figma 文件「🤖座舱组件库-AI版」中，命名为 `icon/<语义名>`，每个 set 含「形态=线形 / 形态=面形」两个 variant。
>
> **维护方式**：用户和 AI 协作维护。每次 AI 帮用户调用图标后，把"业务场景 → 图标"映射回填到本表，让表越用越全。

---

## 使用说明

### AI 查表流程

1. 拿到业务文案/场景关键词（如"驾驶辅助""点赞""刷新"）
2. 在本表"业务场景"列做关键词匹配（同一行多个关键词用 `/` 分隔，命中任一即可）
3. 命中 → 直接用「推荐图标」列的图标名；如有「备选」可一并提供给用户拍板
4. 未命中 → 去 Figma 库搜索（同义词扩展并行搜），仍无果再 `findAll('icon/*')` 拉清单做模糊匹配
5. 用完后把本次确认的映射回填到表中（包括"取舍理由"）

### 形态选择默认规则

- **形态=面形**：list row 主图标、CTA 按钮、需要远观识别的强信息层（默认选这个）
- **形态=线形**：分组标题、辅助操作、密集图标网格中的弱信息层
